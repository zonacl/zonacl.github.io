Términos de uso

Términos

Estos Términos de Servicio se aplican al navegar en todo el sitio web de Zonacl. Como usuario registrado o visitante, ha confirmado el haber leído, entendido y aceptado los siguientes términos y condiciones de estos Términos de Servicio. Si no está de acuerdo con alguno de estos términos, tiene prohibido usar o acceder a este sitio.

Control de usuarios

Comportamiento de los usuarios

Solo debe utilizar el sitio web a través de un navegador web y todas las acciones deben ser realizadas manualmente por el usuario. No se le permite utilizar ningún tipo de software o extensiones de navegador para automatizar las acciones realizadas en el sitio web. No manipulará ningún dato enviado entre su navegador y nuestros servidores. No debe intentar piratear el sistema. Esto incluye, entre otros, intentar acceder, modificar o eliminar datos a los que normalmente no se puede acceder a través de medios habituales. Tampoco debe intentar realizar acciones fuera de las funciones habituales del sitio web.

Cuenta de usuario

El sitio web no pide credenciales personales para registros de cuentas o inicio de sesión, únicamente se pide al usuario información personal en el formulario de contacto. El usuario es responsable de ingresar datos reales y verificar que el formulario es propio del sitio web y no de anuncios o enlaces externos.

Pagos

Los usuarios son responsables de seleccionar el método de pago que deseen y de la información del mismo que proporcionen. Todos los pagos serán procesados de acuerdo a la información proporcionada por el usuario. En el caso de que falle un pago y o se cancele, las tasas que se apliquen incurrirán en el usuario.

Todos los pagos, serán enviados y entregados instantáneamente por el procesador de pagos seleccionado.

No somos responsables de las acciones de nuestros usuarios o actividades ilegales que puedan ocurrir por cualquiera de nuestros usuarios utilizando fondos o criptomonedas donadass o retiradas de nosotros.

Anuncios

Todos los sitios anunciados se muestran dentro de un iFrame y es responsabilidad del anunciante el verificar que el sitio siendo anunciado funciona y se muestra dentro de un iFrame según lo esperado. Aceptamos cualquier tipo de anuncio excepto páginas que se salgan de los Frames, tengan código malicioso, se redirijan a otras páginas, no sirvan todo el contenido por HTTPS, tengan contenido para adultos, contenido ilegal, tengan ofertas multi-nivel, vendan referidos, incluyan juego o apuestas, contenido sobre juego/apuestas o pertenecientes a empresas de juego/apuestas. Por "juego" se entiende todo tipo de juegos de azar y derivados.

Modificaciones

Podemos modificar estos términos de servicio para nuestro sitio web en cualquier momento con o sin previo aviso. Al utilizar este sitio web, acepta estar sujeto a la versión actualizada de estos términos de servicio.

El idioma principal del sitio y los terminos de servicio son publicados en español y es obligación del usuario encontrar la manera de traducirlos e interpretarlos, por lo tanto no se permitiran excusas de desconosimiento de dichos terminos por cuestiones de idioma.
